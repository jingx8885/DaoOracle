# 思考，快与慢（5）：那个我们都曾拥有却渐渐失去的世界

# 一个深夜读书人的"啊哈时刻"

最近半夜读《思考，快与慢》的第五部分，突然被一个概念击中了——两个自我！我放下书，看着窗外的夜色，心里涌起一种说不出的共鸣。

这个想法让我兴奋得睡不着觉（又给熬夜找了个完美的借口😅）。

# 住在我们心里的两个"房客"

你知道吗？卡尼曼老爷子在《思考，快与慢》里提出了一个特别有意思的观点——我们每个人其实都有两个自我。而这两个自我，恰好对应着他更为人熟知的"快思维"和"慢思维"：

**第一个是"体验自我"**：它活在当下，只关心此时此刻的感受。这就是我们的"快思维"——直觉的、感性的、本能的。就像一个专注的孩子，完全沉浸在正在做的事情里。当你咬下一口美味的蛋糕，感受到甜腻在舌尖绽放的那一瞬间，这就是体验自我在工作。它不会去想"这块蛋糕会让我发胖"，也不会回忆"上次吃蛋糕是什么时候"，它就是纯粹地、感性地享受着当下的美好。

**第二个是"记忆自我"**：它活在故事里，总是在回忆过去、规划未来、构建意义。这就是我们的"慢思维"——理性的、分析的、深思熟虑的。它像一个勤奋的编剧，不断地把我们的经历编织成连贯的故事。同样是那块蛋糕，记忆自我会理性地分析："这提醒我想起了小时候生日时妈妈做的蛋糕"，或者计算得失："我明天开始减肥，今天就最后放纵一次"。

有意思的是，体验自我更像是我们的"感性大脑"——它追求快乐，逃避痛苦，活在情感的波动中；而记忆自我更像是我们的"理性大脑"——它要意义，要逻辑，要一个说得通的人生故事。

这两个"房客"就这样住在我们心里，一个感性冲动，一个理性计算，每天都在上演各种小剧场。有时候它们和睦相处，有时候又会闹得不可开交。

# 当两个室友开始打架

先说说这两个自我是怎么"养成"的吧。

**记忆自我**，我喜欢叫他"学霸室友"。这家伙特别理性，成长路径清晰得像教科书：记忆→知道→理解→运用→掌握→熟练→外化。他擅长跟客观世界打交道，喜欢用文字学习，越学越聪明，能力越来越强，就像给自己装了很多"技能包"。

**体验自我**，我更愿意叫他"艺术家室友"。这位就感性多了，成长路径是：感触→共鸣→体验→感悟→智慧→内化。他专门负责跟那个主观的、感性的世界互动，不靠文字，全凭感觉，越体验越有智慧，最后能改变我们内在的习惯和性格。

**咬文嚼字：让我们仔细品味这两条路径**

**理性自我的成长路径：**
- **记忆**：记住并能够回忆，就是简单的回放
- **知道**：明白路在哪里，但你可能没走过  
- **理解**：能按一定的道理把东西解开，具备了拆解的能力
- **运用**：开始能在实际情况中使用
- **掌握**：能用自己的手牢牢握住
- **熟练**：能对它应用自如
- **外化**：就像你长多了一只手一样，对外的能力提升了

**感性自我的成长路径：**
- **感触**：用你的感觉触碰
- **共鸣**：能做到感同身受
- **体验**：整个身心都能沉浸其中  
- **感悟**：用你的感知悟到了（对标理性的"掌握"）
- **智慧**：形成了深层的洞察力
- **内化**：修改了我们的内在习惯和性格

你知道聪明人和有智慧的人的区别吗？我觉得聪明的人是"学霸室友"很强大，能快速学习各种技能，很快外化各种能力；而有智慧的人是"艺术家室友"很强大，能快速内化自我，修改自己的底层逻辑。

顺便说一句，我们的性格大部分藏在快思维里，性格就是由一系列习惯组成的快思维模式。我们的贪嗔痴慢疑，也大多藏在快思维里面。

问题来了——这两个室友经常打架！

# 那些让人哭笑不得的"内战"

你有没有遇到过这种情况：

**深夜熬夜现场**
- 学霸室友：「明天还要上班，赶紧睡觉！身体健康最重要！」
- 艺术家室友：「但是这个剧情好精彩啊，我就想再看一集...」
- 最终结果：一边看一边内疚，一边内疚一边看🤦‍♀️

**减肥吃草现场**  
- 学霸室友：「要吃营养均衡的健康餐，蛋白质配纤维素！」
- 艺术家室友：「但是我就想吃那个热腾腾的小笼包...」
- 最终结果：吃着草想着包，满心都是不甘愿😭

**周末学习现场**
- 学霸室友：「应该学习充电，提升自我价值！」
- 艺术家室友：「我只想躺着刷剧，什么都不想干...」
- 最终结果：一边刷剧一边焦虑，一边焦虑一边刷剧😵‍💫

王阳明先生在《传习录》里把这种状态叫"知行不一"，说得特别准：我知道这是对的，但我不想做，我做不到。这就是两个自我打架导致的内耗。

有时候我觉得，承认"我就想熬夜"可能比"我应该早睡"更诚实。因为往往那个感性的自我，才知道我们真正想要什么。


# 古人的智慧：如何让两个室友和好

王阳明早就看透了这个问题，他给出的解决方案特别有意思——知行合一。

但他的方法不是让一个自我消灭另一个，而是让两个自我学会合作。具体怎么做呢？

**第一步：致良知**
说白了就是：承认你的真实感受！不要再自我欺骗了。想熬夜就承认想熬夜，想吃甜食就承认想吃甜食。这是对话的第一步。

往往对内在的问题，感性那个自我的答案才是正确的答案，理性那个自我经常会欺骗我们。比如熬夜这件事，就大方承认"我就想熬夜"，这比"我应该早睡但是..."要诚实得多。这就是对内认知的第一步，也是解决两个自我割裂感的关键。要拒绝虚假的理性包装，直接面对内在的真实选择。我们也要培养一种意识，但说出我知道的时候，要停一停，问问感性的那个自我，是否知道，他知道才是真的知道。

**第二步：破三个贼——遵循易经的阴阳转化逻辑**

这个过程特别有意思，符合易经里面的变化规律：两个自我在对立的时候，其实是一个此消彼长的动态过程。感性自我（艺术家室友）从强到弱，理性自我（学霸室友）从弱到强，这就是易经里讲的阴阳互转、刚柔相济。

**第一阶段：坐中静，破焦虑之贼（感性极强，理性极弱）**
这时候"艺术家室友"无比强大，"学霸室友"无比弱小，但两者在激烈打架，所以产生躁动和焦虑。

这种焦虑我们可以打个比喻：艺术家室友就像是一根粗大的铁棒，而学霸室友则是一个小石子。你拿着小石子在不断打磨铁棒，铁棒有点发烫，但是没有任何变化，石子却可能先碎掉。这就是那种"我知道应该这样，但就是做不到"的无力感。

这正符合易经"阳盛至极，必生阴机"的道理。体验自我（阳）在最强大的时候，其实也是转机的开始。越是躁动不安，越说明变化即将来临。

具体做法就是静下来，不急着行动，也不急着批判，静静观察这场"内战"。静是慢思维的力量来源，让理性的力量慢慢生长，这样才能给记忆自我更大更坚硬的"石头"。当内心真正静下来的时候，你会发现，那根"铁棒"其实没有想象中那么坚硬。

**第二阶段：舍中得，破欲望之贼（两自我势均力敌）** 
这时候两个自我势均力敌，正在拉锯战，这是转化的关键时刻。就像太极图中间那条S形的分界线，阴阳平衡时，最容易产生变化。

这个阶段很微妙，"铁棒"开始变得柔软一些，"石子"也变得坚硬一些。你会发现原来那些"不可违背"的欲望和习惯，其实是可以商量的。比如原来要刷三小时手机，现在可以跟自己说："要不我们就刷两小时？"

具体做法就是说服"艺术家室友"适当放下一些执着，不是压制，而是像哄小孩一样的商量协调："本质就是放下一些性格和习惯"。这是快思维愿意妥协的时刻，也是新习惯的种子。

这正是易经中"舍中得"的智慧——看似是失去，实际是获得。舍掉一些旧的执着，才能得到新的自由。

**第三阶段：事上练，破犹豫之贼（理性渐强，感性渐弱）**
这时候"学霸室友"开始占上风，但"艺术家室友"还在犹豫不决，就像是在说："真的要改变吗？我还有点舍不得..."

这个阶段符合易经"阴盛阳衰"的规律，体验自我的力量在减弱，理性自我的力量在增强，正是行动改变的最佳时机。就像古人说的"趁热打铁"，这时候必须行动起来！

行动的目的不是强迫，而是创造新的体验，让身体形成新的记忆。比如早起一次，让身体感受到清晨的美好；健身一次，让身体体验到运动的快感。只有新的体验才能真正说服感性自我："原来这样也挺好的啊！"

这样，新的习惯回路就开始形成了。"艺术家室友"从抗拒到接受，再到主动配合，这就完成了一次完整的自我改变过程。

这个过程就像易经里的"复卦"——一阳来复，生生不息。从混乱到和谐，从对立到统一，遵循的是自然的变化规律，不是强制的压制，而是顺势的引导。

# 那个我们都曾拥有的神奇世界

说到这里，我想起小时候的一些奇妙体验。

你还记得小时候那种完全沉浸在游戏里的感觉吗？那时候，一个纸箱子能变成太空飞船，一根木棍能变成魔法棒，几块石头能变成做饭的锅碗瓢盆。那个世界里，时间不存在，逻辑不重要，只有纯粹的体验和快乐。

那个世界，你无法用文字准确描述，却能真真切切地感受到。就像《道德经》说的"道可道，非常道"，《心经》说的"色即是空"的那个"空"的世界。

那是一个以体验和快乐为根本的世界，一个只能靠感悟而不能靠学习才能接触到的世界。

可是长大后，我们渐渐失去了进入那个世界的能力。我们学会了用理性分析一切，用文字记录一切，用逻辑解释一切。我们的"学霸室友"越来越强大，"艺术家室友"却越来越被边缘化。

我们开始活在一个可以被描述、被记录、被量化的世界里，却忘了那个只能被感受的世界。

# 两个自我的成长轨迹

我突然意识到一个有趣的现象：我们每个人的成长过程，其实就是两个自我力量对比变化的过程。

小时候，我们的"艺术家室友"特别强大。还记得那些完全沉浸在游戏里的时光吗？那时候，一个纸箱子能变成太空飞船，一根木棍能变成魔法棒，几块石头能变成做饭的锅碗瓢盆。那个世界里，时间不存在，逻辑不重要，只有纯粹的体验和快乐。

但随着我们长大，"学霸室友"越来越强大，"艺术家室友"却越来越被边缘化。我们学会了用理性分析一切，用文字记录一切，用逻辑解释一切。我们开始活在一个可以被描述、被记录、被量化的世界里，却渐渐失去了那种纯粹感受的能力。

这让我想到了人工智能的局限性。AI再厉害，它也只能处理文字信息，只能在"可描述"的世界里遨游。它永远无法感受到一杯茶的温暖、一首歌的感动、一个拥抱的治愈。

这可能就是为什么，无论人工智能多么发达，它都无法真正成为"人"的原因。因为它缺少了那个"艺术家室友"，缺少了体验和内化的能力。

# 重新找回完整的自己

我们每个人的成长过程，其实就是在这两个世界之间找平衡的过程。

小时候，我们只活在感性的世界里，快乐但缺乏方向。
长大后，我们主要活在理性的世界里，有目标但少了灵魂。

最好的状态，应该是两个世界都能自由穿梭，两个自我都能和谐共处。

既能用理性的力量去实现目标、解决问题，也能用感性的力量去体验生活、感受美好。既能在"可见"的世界里建功立业，也能在"不可见"的世界里滋养心灵。

也许，这就是所谓的"知行合一"的真正含义——不是知道了就能做到，而是让知道的那个自我和感受的那个自我，学会对话，学会合作，学会在两个世界之间自如地游走。

# 结语：给自己和世界一个拥抱

最后，我想说：如果你也经常感到内耗，如果你也觉得自己活得有点分裂，别担心，你不是一个人！

我们都在学习如何跟自己的两个室友和平相处，我们都在寻找回到那个美好世界的路径。

也许答案不在书本里，不在别人的建议里，而在我们每一次真诚的自我对话里，在我们每一次勇敢的内心探索里。

愿我们都能重新找回那个完整的自己，愿我们都能在两个世界之间自由穿梭。

愿每个人的"学霸室友"和"艺术家室友"都能成为最好的朋友！💕
